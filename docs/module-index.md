> **Purpose.** This file is the authoritative map of every server endpoint, library module, script and frontend surface in the app. It exists to stop two failure modes: (1) **building a feature that already exists**, and (2) **guessing at what a file does instead of reading it**.
>
> **Rule before you build.** Before creating any new API route, lib module, or client script, **grep this file first** for the capability (e.g. `sales`, `competitor`, `commitment`, `sensitivity`). If a row already covers it, extend that file — do not create a parallel one.
>
> **Rule for truth.** Every row below was generated by reading the actual file: HTTP handlers are the real `export const GET/POST/...`; data sources are the literal BigQuery table/view identifiers and external hosts found in the code. This file is a **map, not a contract** — when you touch a file, still open it and quote the exact lines. If you change a file's handlers or data sources, update its row here in the same commit.
>
> _Generated 2026-07-09 by reading every file under `src/pages/api`, `src/lib`, `src/scripts`, and the frontend surfaces. Regenerate the same way when it drifts._

## Data access — the truth about how the app reads data

- BigQuery client: [`src/lib/bq.ts`](../src/lib/bq.ts) → `makeBQClient(projectId)`. Credentials from `GOOGLE_SERVICE_ACCOUNT_KEY` (inline JSON) or `GOOGLE_APPLICATION_CREDENTIALS` (keyfile). **The project id is passed per-call by the caller** — it is not hardcoded in `bq.ts`.
- App data project: `muse-square-open-data` (EU). `ms-database-472505` is billing-only — never queried for app data.
- Datasets observed in code: `semantic` (`vw_*` views — the app's read layer), `mart` (`fct_*` facts), `raw` (source/ingested + user state), `analytics` (app-write state: logs, drafts, commitments, channel configs, sensitivity store), `dims` (`dim_*`), `open_data`.
- External services called: Anthropic API (`api.anthropic.com/v1/messages`), Browserless (`production-sfo.browserless.io`), Google Places / Business Profile, Meta Graph, Slack, Resend (email), BestTime, French BAN (`api-adresse.data.gouv.fr`), dbt Cloud (`ym384.us1.dbt.com`).
